# 🚦 Traffic Prediction Using LightGBM & XGBoost

## 📌 Project Overview

This project predicts the **Traffic Situation** using Machine Learning algorithms such as **LightGBM** and **XGBoost**. The model is trained using vehicle count data and time-related features to classify traffic conditions accurately.

---

## 🎯 Objectives

- Predict traffic situation using machine learning.
- Compare the performance of XGBoost and LightGBM.
- Perform data preprocessing and feature engineering.
- Visualize the dataset using EDA.
- Save the trained model for future use.

---

The dataset contains the following features:
